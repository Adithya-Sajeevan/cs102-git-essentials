1. git --version
To check whether git is installed.

2. ssh-keygen -t RSA
To create key pair.

3. ls -la ~/.ssh | grep 'id_rsa'
To check whether our computer already has an SSH key generated by listing files in the hidden folder .ssh.

4. git remote -v
To check whether remote origin is already be setup. 

5. git status
To see current status of the repository.
